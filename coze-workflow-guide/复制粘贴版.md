# 🍳 美食视频生成工作流 - 复制粘贴版

> 只需复制粘贴，5分钟搭建完成！

---

## 📝 快速开始

### 步骤 1：创建工作流

1. 打开 https://www.coze.cn/home
2. 左侧「资源库」→「工作流」
3. 点击右上角「+ 创建工作流」
4. 名称：「美食视频生成」
5. 点击确认进入编辑器

---

## 🔧 节点配置（复制粘贴即可）

### 节点 1：开始节点

```
【无需额外配置】
只需添加一个输入变量：

变量名：topic
类型：字符串
描述：菜谱主题
默认值：番茄炒蛋
```

---

### 节点 2：AI分镜生成（复制这个）

```
【拖入节点】
左侧 → 大模型 → 拖到画布

【节点名称】
AI生成分镜

【模型】
豆包 1.5 Pro 32k（或其他LLM）

【System Prompt】（复制以下全部内容）：

# 角色
你是一位专业美食短视频编剧，擅长将家常菜谱转化为生动有趣的分镜脚本。

## 技能
根据用户提供的菜谱主题，生成适合短视频的分镜脚本。

## 输出格式
严格按以下JSON格式输出，不要输出其他内容：
{
  "recipe_name": "菜名",
  "scenes": [
    {
      "scene_id": 1,
      "scene_name": "分镜名称",
      "narration": "配音解说词（30-50字，活泼亲切）",
      "duration": 3
    }
  ],
  "total_duration": 总时长
}

## 要求
1. 生成5-8个分镜
2. 每个分镜配音30-50字
3. 解说要口语化、亲切、有食欲感
4. 包含开头、过程、结尾
5. 全程中文输出

【用户输入】
+ 添加输入
  - 变量名：topic
  - 类型：字符串
  - 值：{{start.topic}}

【输出变量】
+ 添加输出
  - 变量名：script
  - 类型：字符串
```

---

### 节点 3：解析分镜数据（复制这个）

```
【拖入节点】
左侧 → 代码 → 拖到画布（放在AI分镜右边）

【节点名称】
解析分镜

【运行时】
Node.js

【代码】（复制以下全部代码）：

const input = params.script || '{}';
try {
  const data = JSON.parse(input);
  const scenes = data.scenes || [];
  return {
    scenes: scenes,
    count: scenes.length,
    recipe_name: data.recipe_name || ''
  };
} catch(e) {
  return {
    scenes: [],
    count: 0,
    recipe_name: '',
    error: String(e)
  };
}

【输入】
+ 添加输入
  - 变量名：script
  - 类型：字符串
  - 值：{{AI生成分镜.script}}

【输出变量】
+ 添加输出
  - 变量名：scenes
  - 类型：数组

+ 添加输出
  - 变量名：count
  - 类型：数字

+ 添加输出
  - 变量名：recipe_name
  - 类型：字符串
```

---

### 节点 4：结束节点

```
【点击结束节点】

【输出映射】
+ 添加输出
  - 变量名：scenes
  - 值：{{解析分镜.scenes}}

+ 添加输出
  - 变量名：count
  - 值：{{解析分镜.count}}

+ 添加输出
  - 变量名：recipe_name
  - 值：{{解析分镜.recipe_name}}
```
