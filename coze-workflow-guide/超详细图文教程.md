**📋 代码内容（完整复制）：**

```javascript
const input = params.script || '{}';
try {
  const data = JSON.parse(input);
  const scenes = data.scenes || [];
  return {
    scenes: scenes,
    count: scenes.length,
    recipe_name: data.recipe_name || ''
  };
} catch(e) {
  return {
    scenes: [],
    count: 0,
    recipe_name: '',
    error: String(e)
  };
}
```

## 🔗 第八步：连接所有节点

### 8.1 认识连接点

```
每个节点边缘有小的圆点（●），这是连接点

  开始节点：右侧有 ●（输出连接点）
  结束节点：左侧有 ●（输入连接点）

  ● ─────────────── ●
  ↑ 从这里拖出    ↑ 连接到这里
```

### 8.2 连接步骤

```
连接 1：开始 → AI分镜
  1. 鼠标悬停在「开始」节点上
  2. 看到右侧出现的小圆点 ●
  3. 鼠标按住那个 ● 不放
  4. 拖出一条线
  5. 拖到「AI分镜」节点的左侧
  6. 松开鼠标，线条会吸附上去

连接 2：AI分镜 → 解析分镜
  1. 点击「AI分镜」节点右侧的 ●
  2. 拖线到「解析分镜」节点左侧

连接 3：解析分镜 → 结束
  1. 点击「解析分镜」节点右侧的 ●
  2. 拖线到「结束」节点左侧
```

**连接完成后的工作流：**

```
  ┌─────┐     ┌─────────┐     ┌─────────┐     ┌─────┐
  │     │     │         │     │         │     │     │
  │ 开始 │────▶│ AI分镜  │────▶│ 解析分镜│────▶│ 结束 │
  │     │     │         │     │         │     │     │
```
